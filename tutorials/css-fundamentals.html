1. CSS Basics

CSS (Cascading Style Sheets) controls the appearance of HTML elements:

/* CSS Syntax */
selector {
    property: value;
}

/* Example */
h1 {
    color: blue;
    font-size: 24px;
}

2. Selectors

Element Selector

p { color: red; }

Selects all paragraph elements

Class Selector

.highlight { background: yellow; }

Selects elements with class="highlight"

ID Selector

#header { font-weight: bold; }

Selects element with id="header"

3. Box Model

.box {
    width: 200px;
    height: 100px;
    padding: 20px;
    border: 2px solid black;
    margin: 10px;
}
