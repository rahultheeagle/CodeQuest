Article Structure

Proper use of <article> element for standalone content:

<article role="article" aria-labelledby="title">
  <header>
    <h1 id="title">Article Title</h1>
    <time datetime="2024-01-01">January 1, 2024</time>
  </header>
  <div class="content">Article content...</div>
  <footer>Author information</footer>
</article>

Navigation Structure

Accessible navigation with proper ARIA attributes:

<nav role="navigation" aria-label="Main navigation">
  <ul role="menubar">
    <li role="none">
      <a href="#" role="menuitem">Home</a>
    </li>
    <li role="none">
      <a href="#" role="menuitem">Challenges</a>
    </li>
  </ul>
</nav>

Form Accessibility

Accessible form with proper labeling and validation:

<form novalidate aria-labelledby="form-title">
  <fieldset aria-labelledby="basic-info">
    <legend id="basic-info">Basic Information</legend>
    <label for="username">Username</label>
    <input id="username" 
           type="text" 
           required 
           aria-describedby="username-help username-error">
    <small id="username-help">Help text</small>
    <div id="username-error" role="alert"></div>
  </fieldset>
</form>

ARIA Attributes

Proper ARIA usage for enhanced accessibility:

<div role="progressbar" 
     aria-valuenow="75" 
     aria-valuemin="0" 
     aria-valuemax="100" 
     aria-label="Challenge progress">
  <div class="progress-bar" style="width: 75%"></div>
  <span class="sr-only">75% complete</span>
</div>

<button aria-expanded="false" 
        aria-controls="menu" 
        aria-haspopup="true">
  Menu
</button>
